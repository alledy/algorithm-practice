## 풀이법
- 문제가 요구하는 출력은 최적 요금제와 요금이다.
- 따라서 데이터 사용량에 따라 최적 요금제를 계산하는 함수 하나, 요금제와 데이터 사용량을 넣으면 요금을 계산하는 함수 하나를 만든다.

- if else문의 과도한 사용으로 좀 지저분 해 보인다

- 테스트케이스 하나를 틀리는 데 어떤 부분이 문제인지 아직 못찾음
